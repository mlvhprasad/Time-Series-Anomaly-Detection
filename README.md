
# Time-Series Anomaly Detection & Early Warning System

## Overview

This project analyzes district-wise Indian rainfall data to identify unusual rainfall patterns using machine learning-based anomaly detection techniques.

## Dataset

Indian Rainfall Dataset – District-wise Daily Measurements

- Records: 8,790
- Features: 34
- Coverage: Multiple Indian states and districts

## Objectives

- Analyze rainfall trends
- Engineer temporal features
- Detect anomalous rainfall events
- Support early warning systems

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Methodology

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Isolation Forest-based Anomaly Detection
5. Visualization of Detected Anomalies

## Results

Successfully identified unusual rainfall observations that may indicate extreme weather events.
